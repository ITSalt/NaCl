NaCl 2.12.0 — techniques-over-workflows

Мы прогнали Claude Code dynamic workflows на собственном фреймворке и получили прямой вердикт: критик-панель в виде воркфлоу стоит ~15× токенов одиночного сильного агента *с доступом к репозиторию* — за тот же результат. «Победил одиночный агент» на деле означало «победил доступ к репозиторию». Дешёвые и крупные выигрыши — это техники, а не воркфлоу, и лишь один сценарий вообще оправдывал воркфлоу. Этот релиз собирает обе половины.

Что вошло:

— Усиление скиллов (валидировано head-to-head на реальном проекте). Главный выигрыш: review и verify-code теперь ОБЯЗАНЫ трассировать кросс-файл — читать вызываемые функции/рантайм, который производит данные, и потребителей изменённого, а не только дифф. На family-cinema UC033-BE diff-only ревью пропустило настоящий BLOCKER (сервисы «видео» и «музыка» оба звали клиент, генерирующий только картинки — мёртвый конфиг) и ложно зачло требование; ревью с доступом к репо его поймало.

— Трассируемость требований: review/verify-code/qa теперь проверяют каждый критерий приёмки (реализовано? достижимо? покрыто тестом?) — закрывая класс «отсутствующее требование», где в диффе всё верно, а нужное просто не написано.

— Детерминированный классификатор статусов для verify-code (чистая таблица решений + контракт-пин тест 17/17) вместо приоритета в прозе, который агент выводил заново каждый запуск. Словарь статусов прежний — в код переехал только способ вывода.

— «Сомневаешься — оставь» + самосостязательный проход одним агентом: опровергни собственные находки высокой важности, перечитав код; отбрасывай только по позитивному доказательству. nacl-tl-fix теперь перечитывает корневую причину — зелёный тест доказывает, что симптом ушёл, а не что фикс не закрыл лишь один из нескольких носителей дефекта.

— Новое: /nacl-postmortem — единственный воркфлоу, который себя оправдывает. Пять параллельных аудиторов → проверка по доказательствам → детерминированный синтез GAP→владелец-скилл, для разового аудита «какой гейт скилла пропустил этот баг». Провалидировано на размеченной фикстуре и head-to-head на family-cinema: 36 QA-SKIP-ов + 4 кросс-UC находки, которые одиночный агент не раскрыл полностью, при ~7.7× стоимости (тиринг моделей срезал 15× эксперимента почти вдвое). Opt-in (CC ≥ 2.1.154); проза-рецепт остаётся дешёвым дефолтом.

Без ломающих изменений — каждая правка скилла аддитивна и сохраняет выходной контракт; новый воркфлоу и скилл — opt-in. Codex-зеркала обновлены в паритете.
