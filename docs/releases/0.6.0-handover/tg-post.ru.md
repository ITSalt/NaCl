**NaCl 0.6.0: передача графа + исправление изоляции Compose-проектов**

В этом релизе появился инструментарий для передачи графа между машинами. `handover-export.sh` снимает слепок базы Neo4j через APOC, сжимает и шифрует результат через `age`, сохраняет манифест с количеством узлов и связей. `handover-import.sh` расшифровывает архив, проверяет манифест и воспроизводит граф на принимающей машине. Пароль передаётся по отдельному каналу; зашифрованный архив фиксируется в git.

Параллельно закрыта ошибка изоляции контейнеров между проектами. Каждая папка `graph-infra/` ранее получала одинаковое имя Compose-проекта, из-за чего `docker compose up -d --remove-orphans` в одном проекте мог без предупреждения удалить контейнеры и тома данных других проектов. Канонический шаблон теперь содержит уникальное поле `name:`; `nacl-init` записывает `COMPOSE_PROJECT_NAME` в `.env` каждого нового проекта; все восемь существующих проектов переведены на новый шаблон и прошли регрессионное тестирование.
