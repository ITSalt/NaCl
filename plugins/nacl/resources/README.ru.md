# Ресурсы пакета NaCl

Это корень замкнутых ресурсов сгенерированного Codex-плагина. Публичные навыки
находятся в корне плагина; внутренние workflows, справочные материалы, graph
assets и актуальная документация доступны только по путям внутри пакета.

README исходного репозитория намеренно не копируется: его ссылки на checkout
невалидны внутри установленного cache-пакета.
